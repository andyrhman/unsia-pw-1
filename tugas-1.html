Tutorial Menggunakan Jupyter Notebook

Tempat untuk menuliskan autau menjalankan code disebut dengan Cell
Pada Jupyter Notebook terdapat 2 jenis Cell yaitu:

  1. Code Cell
  2. Markdown Cell

Untuk masuk kedalam mode Cell bisa menggunakan command Y dan command M untuk mode Markdown

Menjalankan Cell

Jika ingin menjalankan Cell, kita dapat menggunakan Ctrl+Enter atau dengan Shift+Enter

Menambahkan Cell

Jika kita kehabisan Cell maka dapat ditambahkan menggunakan

  • command A (above) -> Menambah diatas
  • command B (below) -> Menambah dibawah

Tetapi perlu diperhatikan jika ingin menambahkan Cell warna garis Cell disebelah kiri harus berwarna biru yang menandakan kita masuk dalam command mode, selain itu ada juga yang berwarna hijau yaitu edit mode

Shortcut

  • Delete Cell menggunakan command D+D
  • Cut Cell menggunakan command X
  • Paste below menggunakan command V
  • Paste above menggunakan command Shift+V
  • Undo menggunakan command Z

How to print a text

  • Sebagai pengalaman ngoding awal, kita bisa menampilakn sesuatu lewat Python.
  • Pada bagian ini, kita akan coba menampilkan sebuah text.
  • Caranya adalah dengan menggunakan perintah print melalui syntax

  • print "<teks yang ditampilkan>"

  • Misalkan kita mau menampilkan kalimat Helllo World!
  • Kita dapat melakukannya dengan cara seperti berikut

  • print "Hello World!"

  • Selain menggunakan tanda petik dua ("..."), kita juga dapat menggunakan tanda petik satu ('...')

  • print ('Hello world!')

  • Sekarang ayo latihan, coba print kalimat Halo Dunia!
